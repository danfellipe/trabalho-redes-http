HTTP

O que é HTTP?

HTTP é um protocolo que permite a obtenção de recursos, tais como documentos HTML.
É a base de qualquer troca de dados na Web e um protocolo cliente-servidor, o que significa que as requisições são iniciadas pelo destinatário, geralmente um navegador Web. Um documento completo é reconstruído a partir dos diferentes sub-documentos obtidos, como por exemplo:
texto, descrição do layout, imagens, vídeos, scripts e muito mais.

Clientes e servidores se comunicam trocando mensagens individuais (em oposição a um fluxo de dados). As mensagens enviadas pelo cliente, são chamadas de requisições (requests), e as mensagens enviadas pelo servidor como resposta são chamadas de respostas (responses).
